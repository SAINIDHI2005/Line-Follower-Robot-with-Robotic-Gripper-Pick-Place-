# 🚗 Line Follower Robot with Robotic Gripper (Pick & Place)

This project presents a **compact line follower robot integrated with a robotic grabber**, designed as a proof-of-concept for **autonomous pick-and-place operations**. The system combines line tracking with object handling, making it suitable for applications like **warehouse automation, material transport, and smart robotics systems**.

---

## 🧠 Project Overview

The robot follows a predefined path using line sensors and is equipped with a **servo-driven robotic gripper** capable of picking and placing objects at designated locations. The design focuses on simplicity, modularity, and real-world adaptability.

---

## ⚙️ Design & Modeling

- Entire assembly designed in **Autodesk Fusion 360**
- Custom chassis and gripper mechanism modeled from scratch  
- Standard components integrated using imported STL files:
  - N20 geared motors  
  - Motor brackets  
  - IR sensor module  
  - Servo motor  

The design ensures proper alignment, compactness, and ease of assembly.

---

## 🔩 Key Components

- **N20 DC geared motors** for differential drive  
- **IR sensor array** for line detection  
- **Servo motor** for gripper actuation  
- **Motor driver module** (e.g., L293D / TB6612FNG)  
- **Custom-designed chassis**  
- **Robotic gripper mechanism**
